1：首先安装node js --->node-v14.2.0-x64.msi

2:执行命令 npm install -g vue-cli 开始安装vue-cli

3:安装完成执行命令 vue -V 可以查看当前vue的版本，表明安装成功

4:执行名 vue init webpack vue-demo 创建vue项目 vue-demo是项目名，在这之中可以根据提示进行

5:如果你选择手动安装，在控制台界面根据提示进行操作一下命令即可

   cd vue-demo
   npm install (or if using yarn: yarn)
   npm run lint -- --fix (or for yarn: yarn run lint --fix)
   npm run dev


支持vue的脚手架安装完成 。
